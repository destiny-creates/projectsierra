## Project Description

This project is a fully automated pentesting suite, with only 3 things required by the end user. A selection of wether you want to use the recon, or exploitation module. With this, the tool automatically uses programs like nmap, and dnsdumpster, to pull as much information as possible including vulnerabilities, from your target, and also automates and simplifies the process of payload creation, AV evasion, and much more. This tool is used and tested on hackerone, hackthebox.eu, and more!
